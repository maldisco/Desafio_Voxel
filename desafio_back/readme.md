## Consulta de API's e manipulação de dados

## Dependências
- SQLAlchemy
- Requests

## Utilização
### ⚙️ Como Executar o Programa

#### Primeiramente coletar os dados da API e criar o banco de dados.

```
python3 ./db.py
```
ou
```
py ./db.py
```

#### Em seguida, executar o arquivo principal.

```
python3 ./main.py
```
ou
```
py ./main.py
```
